# Taylor Swift Albums Data Analysis

## Sobre o projeto

Este projeto realiza uma análise exploratória de dados da discografia da Taylor Swift utilizando Python, Pandas e Google Looker Studio.

Os dados foram obtidos a partir de datasets públicos contendo informações sobre músicas, álbuns e métricas do Spotify.

---

## Objetivos

- Praticar análise de dados com Python
- Realizar limpeza e transformação de dados
- Criar visualizações interativas
- Explorar métricas musicais da discografia da Taylor Swift

---

## Tecnologias utilizadas

- Python
- Pandas
- Google Looker Studio
- Jupyter Notebook

---

## Análises realizadas

- Popularidade média por álbum
- Top músicas mais longas
- Quantidade de músicas por álbum
- Conversão de duração das músicas
- Rankings de popularidade

## Como executar

1. Clone o repositório

```bash

```

2. Instale as dependências

```bash
pip install pandas
```

3. Execute o notebook

```bash

```
